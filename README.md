# Monospace Icons for VS Code

![](assets/icons.png)

## Usage

1. Open VS Code.
2. Open the command palette (<kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>) and enter _Preferences: File Icon Theme_, _Install Additional File Icon Themes_.
3. Search for the _Monospace Icon Theme_ extension and install.
4. Enter _Preferences: File Icon Theme_ in the command palette and select the Monospace Icon theme in the dropdown.
